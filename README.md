# UNbLOCK

UNbLOCK is a tool designed to assist writers in generating creative prompts to kickstart their writing process. Whether you're struggling with writer's block or simply looking for inspiration, UNbLOCK provides a variety of prompts across different categories to help ignite your creativity.

## Features

- **Prompt Generation**: Choose from various categories such as dialogue, fiction, nonfiction, and poetry to generate prompts tailored to your preferences.
- **Interactive Interface**: User-friendly interface with options to select categories and generate prompts seamlessly.
- **Customization**: Easily customize prompt generation by selecting specific categories or refreshing for new prompts.
- **Responsive Design**: Designed to work smoothly across different devices and screen sizes.


## Getting Started

To run UNbLOCK locally, follow these steps:

1. Clone this repository to your local machine.
2. Navigate to the `backend` folder and install dependencies by running `npm install`.
3. Start the backend server by running `npm start` or `node server.js`.
4. In a separate terminal, navigate to the `frontend` folder and install dependencies by running `npm install`.
5. Start the frontend development server by running `npm start`.
6. Open your web browser and visit `http://localhost:3000` to access UNbLOCK.

## Technologies Used

- **Backend**: Node.js with Express
- **Frontend**: React.js
- **Routing**: React Router
- **Styling**: CSS
